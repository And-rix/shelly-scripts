Older (Gen1) Shelly devices (such as the classic Shelly 1, Shelly 2.5, or Shelly Plug S) do not support **on-device scripting** and are **NOT compatible** with the scripts provided in this repository.

## How-To use:
1. **Create a new script** in your Shelly Plus Plug > [http://shelly-ip/#/scripts](http://shelly-ip/#/scripts)
2. **Copy & Edit** the script you want to run
3. **Save and Start** the script
4. _Useful_ Activate the "***Run on startup***" option
